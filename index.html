Install Git

git is a distributed version control system used to track project changes.

You can download it for your OS at https://git-scm.com/downloads. This site also hosts documentation and tutorials in numerous languages for those new to using git.

Git Setup

Once git is installed, you should set your user name and email

This is done by typing the following (with your information substituted) at the command line (without the $)

$ git config --global user.name "John Doe" $ git config --global user.email johndoe@example.com
Create a Github Account

A github account can be created on the github website.

Account setup and verification is fairly straight forward.
